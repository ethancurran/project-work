Game 2 - Spell the Object

The second interactive game that was implemented into my website was a spelling game. Here the game provided the user with a image and they have to guess what the image is and spell it correctly. All letters used are
recorded within either a correct or incorrect box.


Here within the Javascript a new aspect was used here which was the math.floor. The Math. floor() function returns the largest integer less than or equal to a given number. Here we want too only accept the correct
word which was called above.


The next step within this was then to retrive the users input letter and position this within the correct or incorrect catergory. By adding a event listener for each letter typed. Then through the use of a IF
statement then this sections the users input within the correct category.


The final aspect here then within the Javascript was replacing the users input for the underscore so they can spell out the correct word on screen. Again, this was done throught the use of a IF statement to ensure
corect letters were only selected. From here when all letters are selected the user is promted with a message that they win.

Game 3 - Jumbled Nouns

The final interactive element added to the assessment was a drag and drop game. Here this was layout out in the form of asking the user a question with a series of jumbled letters
and asking to answer by solving the letter into the correct format.

Within the Javascipt the first step taken within the HTML code we had to ensure that the letters called were "draggable="true". This allowed the letters to be draggable rather
than stationary. When Dealing with drag and drop within Javascipt there is 7 aspects to cover in order to have this working correctly. Here these are:
dragstart, drag, dragenter, dragleave, dragover, drop, dragend.

Here Dragstart was adding an effect on the letters when moving allowing the user to understand what letter is going where. Here doing this we called the opacaity of the letter being
dragged to 0.5 to allow the user to see letter behind to place the selected letter.Dragend was the same aspect of when the dragging motion was completed the opacity went back to 1
which is is the default opacity.

With Dragover here the use of e.preventDefault() was added here ot prevent the browsers default behaviour.. in this instance to place the selected letter back to it first location. The drop
aspect has been added with the use of an if statement for the purpose of allowing the dragged item to land on the new desination requested. Now that this has been added
it allows the user to swap letters as they wish.

Finally adding the querySelectorAll for the div class for both the overall div and the letter div this then selects this code for the jacascript to take effect.
