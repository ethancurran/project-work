What to Expect

Here at Love Literature we strive to help children of any age groups find the benefits of learning literature

Here we thought the best way to do this and keep you engaged is by adding in games to help with your learning experience.

We offer 3 informative games giving you the user and fun and learning experience all at the same time!!!



The first Interactive game you will face is a verb quiz. A verb is a word used
to describe an action, state or occurrence. Verbs can be used to describe an action,
that’s doing something. For example, like the word ‘jumping’ in this sentence:

The rabbit was jumping in the field.

They can also be used to describe a state of being, that’s feeling something. For
example, the word ‘likes’ here:

The monster likes rollercoasters.

Or a verb can be used to describe an occurrence, that’s something happening. For
example, the word ‘became’ in this sentence:

The caterpillar became a butterfly.

Why Dont you have a go yourself and test your knowledge?

Below there is a green (correct) box and red (incorrect). any letter that is submited

The final interactive game that will test your knowledge is the jumbled
nouns game.

A noun is a naming word. It names a person, an animal, a place or a thing.
It can also be an abstract concept (something which is not a physical object)
such as a feeling or a state.

There are 6 questions here in total. By now you now how the game goes they
start of easier then get more difficult. Dont worry we know you will beat this game no problem!

Because you are so skilled and made it this far we also added in both words
and images to test you abilities and see if your skills have what it takes.
